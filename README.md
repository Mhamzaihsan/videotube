

1. **Clone the repository**:
   ```bash
   git clone https://github.com/pritambose0/Videotube.git
   cd Videotube
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run the backend**:
   ```bash
   npm start
   ```

5. **Run the frontend**:
   ```bash
   cd client
   npm start
   ```
